# thinkpad-l590-opencore
Opencore EFI for Thinkpad l590

## Tested on macOS 15.7.3 (Sequoia)
 - Thinkpad L590
 - **GPU**: Intel UHD Graphics 620 
 - **CPU**: Intel(R) Core(TM) i5-8265U CPU @ 1.60GHz (Whiskey Lake)


## What works
 - Display, hardware acceleration
 - Wifi via HeliPort
 - Ethernet
 - USB devices
 - USB-C Video
 - Sleep
 - ??

## What doesn't work
 - Bluetooth
 - Fans seem to not be spinning, however the laptop does not overheat.
 - Some Apple things probably, but I don't care
 - Sound
 - Brightness control
 - ??
